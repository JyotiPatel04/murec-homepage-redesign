# MUREC Homepage Redesign

A premium, modern, and responsive homepage redesign of the MUREC website built as part of a Frontend Developer Assessment.

## 🌐 Live Demo

https://murec-homepage-redesign.vercel.app

## 📂 GitHub Repository

https://github.com/JyotiPatel04/murec-homepage-redesign

---

## 📖 Project Overview

This project is a complete redesign of the MUREC homepage while preserving the original brand identity and content.

The goal was to create a premium, elegant, and modern user experience inspired by the visual design language of the Timeless website without copying its layout.

The redesign focuses on:

- Premium editorial layout
- Modern typography
- Better visual hierarchy
- Responsive experience
- Smooth animations
- Clean React architecture
- Improved usability

---

## 🎯 Assignment Objective

- Use the original content from MUREC
- Take design inspiration from Timeless
- Build an original UI
- Create a fully responsive website
- Follow clean frontend development practices

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/JyotiPatel04/murec-homepage-redesign.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
